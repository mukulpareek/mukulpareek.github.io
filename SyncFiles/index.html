    Sync Files

    Sync files across two directories

    Both the folders are provided as parameters to the script, one after the other. Then the script is run on the command prompt. Or with appropriate mods the below can be run from Jupyter as well. This can be converted to an exe file using pyinstaller.

    os.walk(folder_name) is a very useful command, it returns a list of tuples. It walks through the provided folder name, and all its subdirectories. Then for each subdirectory it traverses, it returns a tuple with three elements:

    1. The directory of the folder it is in,
    2. The subdirectories it found underneath, and
    3. The files it found in that folder
    import os
folder = r'C:\Users\user\Documents\mkdocs'
files = [(r, d, f) for r, d, f in os.walk(folder)]

    files[0]

    ('C:\\Users\\user\\Documents\\mkdocs', ['docs', 'site'], ['mkdocs.yml'])

    This shows us that in the directory 'C:\\Users\\user\\Documents\\mkdocs', it found two sub-directories, ['docs', 'site'], and one file ['mkdocs.yml']

    len(files)

    11

    Code below shows how to sync two folders

    import sys
import os
import shutil
print('\n \n Number of arguments:', len(sys.argv), 'arguments.\n \n')

# sys.argv contains all the parameters received from the command line as a list.  Each parameter can be accessed using standard list notation.

if len(sys.argv)==3:
    first = sys.argv[1]
    second = sys.argv[2]
    print('First folder is ', first)
    print('Second folder is ', second)
    #first = 'C:\\Users\\user\\Documents\\mkdocs2\\docs'
    #second = 'C:\\Users\\user\\Documents\\mkdocs\\docs'
    files = [(r, d, f) for r, d, f in os.walk(first)]
    set1 = files[len(files) - 1][2]
    src1 = files[len(files) - 1][0]
    files = [(r, d, f) for r, d, f in os.walk(second)]
    set2 = files[len(files) - 1][2]
    src2 = files[len(files) - 1][0]
    first_to_second = list(set(set1) - set(set2)) # copy these from first to second
    second_to_first = list(set(set2) - set(set1)) # copy these from second to first
    print(len(first_to_second), ' files to copy from first to second')
    print(len(second_to_first), ' files to copy from second to first')
    for x in first_to_second:
        shutil.copyfile(os.path.join(src1,x), os.path.join(src2,x) )
        print('Copied from first to second ', str(os.path.join(src1,x)))
    for x in second_to_first:
        shutil.copyfile(os.path.join(src2,x), os.path.join(src1,x) )    
        print('Copied from second to first ', str(os.path.join(src2,x)))

else:
    print('Type the folder names in the usual Windows format, without quotes, \n'
         'eg, python mysync C:\\Users\\user\\Documents\\mkdocs C:\\Users\\user\\Documents\\mkdocs2. \n'
         '\n',
         'This script will sync the files in the two folders (hopefully) \n \n',
         'You are seeing this message because 2 folder names were not received.')
